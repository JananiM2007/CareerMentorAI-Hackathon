# CareerMentorAI

An AI-powered Career Guidance Platform that analyzes resumes, predicts suitable career paths, identifies skill gaps, recommends courses and internships, and generates personalized career roadmaps.

---

## Project Overview

CareerMentorAI helps students and job seekers make informed career decisions by leveraging Machine Learning and Natural Language Processing (NLP).

The platform analyzes uploaded resumes and provides:

* Career Prediction
* ATS Score Evaluation
* Resume Strength Analysis
* Skill Extraction
* Skill Gap Detection
* Course Recommendations
* Internship Recommendations
* Personalized Career Roadmap

---

## Features

### Career Prediction

Predicts the most suitable career domain based on resume content.

### ATS Score Analysis

Evaluates resume quality using ATS-inspired scoring techniques.

### Resume Score

Provides an overall assessment of resume strength.

### Skill Extraction

Automatically identifies technical and professional skills from resumes.

### Skill Gap Analysis

Highlights missing skills required for the predicted career path.

### Course Recommendations

Suggests relevant learning resources to bridge skill gaps.

### Internship Recommendations

Recommends internships aligned with the predicted career domain.

### Personalized Career Roadmap

Generates a step-by-step roadmap for career growth.

## 📊 Workflow

1. Upload Resume (PDF)
2. Extract Resume Text
3. Clean and Preprocess Data
4. Convert Text using TF-IDF
5. Predict Career Domain
6. Extract Skills
7. Analyze Skill Gaps
8. Calculate ATS and Resume Scores
9. Recommend Courses and Internships
10. Generate Career Roadmap

##  Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/CareerMentorAI.git
```

Navigate to project folder:

```bash
cd CareerMentorAI
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
